## Frequently Confused

| Acronym | Often Confused With | Difference |
|---------|---------------------|------------|
| CRM | CRS | CRM = Customer Relationship Management; CRS = Central Reservation System |
| PMS | POS | PMS = Property Management System; POS = Point of Sale |
| API | UI | API = Application Programming Interface; UI = User Interface |
| SSL | TLS | SSL is older; TLS is the modern version (though often still called SSL) |
| AR | AP | AR = Accounts Receivable (money owed TO you); AP = Accounts Payable (money YOU owe) |
| RevPAR | ADR | RevPAR = Revenue Per Available Room; ADR = Average Daily Rate |
